# Zdjęcia testowe

Przykładowe zdjęcia roślin używane do testów rozpoznawania CLIP (notatnik
`notebooks/demo_plantcare.ipynb`). Źródła: Wikimedia Commons i Openverse (Flickr),
przeskalowane do maks. 512 px.

| Plik | Gatunek |
|------|---------|
| `monstera.jpg` | Monstera deliciosa |
| `sansevieria.jpg` | Sansewieria / wężownica (Sansevieria) |
| `aloe.jpg` | Aloes (Aloe vera) |
| `pothos.jpg` | Epipremnum / pothos (Epipremnum aureum) |
| `orchid.jpg` | Storczyk (Phalaenopsis) |

Aby dodać własne zdjęcie: wrzuć plik do tego katalogu i wskaż go w aplikacji
(`streamlit run app.py`) lub w notatniku.
